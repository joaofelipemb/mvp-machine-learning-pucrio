## Visão geral

Este projeto desenvolve, avalia e discute criticamente um modelo de Machine Learning para identificar transações fraudulentas de cartão de crédito. O trabalho é um relatório técnico executável, que percorre todo o fluxo de um projeto de ML: definição do problema, carga e entendimento dos dados, preparação, modelagem, otimização, avaliação e conclusão.

O foco não é apenas o desempenho bruto, mas a coerência metodológica: tratamento adequado do desbalanceamento de classes, prevenção de vazamento de dados, avaliação temporal honesta e análise crítica dos limites da solução.
